3. Update Operation:

// Update color of a car
db.cars.updateOne(
  { regNo: "KA01AB1234" },
  { $set: { color: "White" } }
);

// Update multiple cars:
db.cars.updateMany(
  { brand: "Toyota" },
  { $set: { warranty: true } }
);


====================================================================

4. Delete Operation:

// Delete one:
db.cars.deleteOne({ regNo: "KA05CD5678" });

// Delete all cars older than 2019:
db.cars.deleteMany({ year: { $lt: 2019 } });
